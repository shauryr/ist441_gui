## IST 441 GUI template

Here are some files/folders in which you might want to make changes - 

- To change $search()$ function:
```sh
seer/views.py
```

- Make changes in the html page:
```sh
seer/templates/seer/results.html
```


- To change images:
```sh
seer/static/img
```
